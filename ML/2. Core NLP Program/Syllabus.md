This document keeps the PDF's exact chapters, topics, tools, and projects, sequenced into 10 days — no topics added, none dropped. **Depth is not determined by the PDF** — the PDF only supplies the topic list; actual depth per topic is decided when each day's content is written, per `ML/CLAUDE.md`'s shared bar.
